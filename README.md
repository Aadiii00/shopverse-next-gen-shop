## Features
- Responsive homepage with hero and product features
- Product listing, categories, deals, and profile pages
- Sidebar dashboard + AI chat window
- Authentication & DB via Supabase (or optional)
- Backend functions with Cursor.sh (or your chosen server)
- Cart & order flow (client-side + persisted to Supabase)
- Accessibility and dark-mode toggle

## Tech stack
- Frontend: React, Vite or Next.js (specify), Tailwind CSS
- Backend: Cursor.sh functions (serverless) or Node/Express on your chosen provider
- Database & Auth: Supabase (Postgres + Auth)
- Hosting: Vercel (recommended), Netlify, or static host
- Optional: GitHub Actions for CI
